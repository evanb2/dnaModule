# DNA Sequencer Module for Drupal 7

### Evan Butler

###### June 8, 2015

#### Description

This module lets a user enter a DNA strand and returns the matched strand.

#### Set up

1. `git clone` the repository into your sites/all/modules folder
2. enable the module
3. enjoy
